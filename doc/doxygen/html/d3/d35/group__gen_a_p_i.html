enum NC_VERB_LEVEL

Verbosity levels.

Enumerator:
NC_VERB_ERROR 

Print only error messages.

NC_VERB_WARNING 

Print error and warning messages.

NC_VERB_VERBOSE 

Besides errors and warnings, print some other verbose messages.

NC_VERB_DEBUG 

Print all messages including some development debug messages.
